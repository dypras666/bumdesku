SHEET 7: PANDUAN PENGGUNAAN

PANDUAN PENGGUNAAN TEMPLATE MANUAL BACKUP BUMDES

1. CHART OF ACCOUNTS (Sheet 1):

  • Berisi daftar lengkap akun sesuai sistem BUMDES
  • Jangan mengubah kode akun yang sudah ada
  • Untuk menambah akun baru, ikuti format kode yang sama
  • Pastikan saldo awal balance (Total Aset = Total Kewajiban + Modal)

2. TEMPLATE JURNAL UMUM (Sheet 2):

  • Gunakan untuk mencatat transaksi harian
  • Pastikan setiap transaksi memiliki debit dan kredit yang seimbang
  • Format kode transaksi: IN/OUT + YYYYMMDD + 4 digit nomor urut
  • Contoh: IN202501010001 (Pemasukan tanggal 1 Jan 2025 nomor 1)

3. BUKU BESAR (Sheet 3):

  • Salin data dari Sheet 2 ke Sheet 3 untuk posting
  • Setiap baris jurnal menjadi satu baris di buku besar
  • Gunakan kode entry format: GL-YYYYMMDD-4 digit nomor urut
  • Status posting: "Posted" untuk transaksi yang sudah final

4. NERACA SALDO (Sheet 4):

  • Otomatis menghitung saldo dari buku besar menggunakan rumus SUMIF
  • Periksa balance: Total Debit harus sama dengan Total Kredit
  • Jika tidak balance, periksa kembali jurnal dan posting
  • Rumus otomatis menghitung saldo akhir berdasarkan normal balance akun

5. LAPORAN LABA RUGI (Sheet 5):

  • Otomatis menghitung dari neraca saldo
  • Menampilkan pendapatan dikurangi beban
  • Hasil akhir adalah laba/rugi bersih periode berjalan
  • Update periode dan nama desa sesuai kebutuhan

6. NERACA (Sheet 6):

  • Otomatis menghitung dari neraca saldo
  • Menampilkan posisi keuangan: Aset = Kewajiban + Modal
  • Periksa balance check di bagian bawah
  • Laba tahun berjalan otomatis diambil dari laporan laba rugi

TIPS PENGGUNAAN:

  • Backup file ini secara berkala
  • Gunakan format tanggal YYYY-MM-DD untuk konsistensi
  • Selalu periksa balance setelah input transaksi
  • Simpan dalam format Excel (.xlsx) untuk mempertahankan rumus
  • Buat copy untuk setiap periode (bulanan/tahunan)

VALIDASI BALANCE:

  1. Total Debit = Total Kredit di Jurnal Umum
  2. Total Debit = Total Kredit di Neraca Saldo
  3. Total Aset = Total Kewajiban + Modal di Neraca

BACKUP BERKALA:

  • Harian: Backup file setelah input transaksi
  • Mingguan: Buat copy file dengan nama berbeda
  • Bulanan: Arsipkan file bulan sebelumnya
  • Tahunan: Buat file baru untuk tahun baru

TROUBLESHOOTING:

  • Balance tidak seimbang: Periksa jurnal dan pastikan setiap transaksi balance
  • Rumus error: Pastikan referensi sheet dan cell benar
  • Data hilang: Restore dari backup terakhir
